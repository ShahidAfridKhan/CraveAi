# SHAHID RESTRO - Restaurant Dashboard

A modern, responsive restaurant dashboard that displays menu items from a Google Sheet in real-time. Built with HTML, CSS, and JavaScript.

## 🍽️ Features

- **Real-time Data**: Fetches menu data directly from Google Sheets
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Category Filtering**: Filter items by Veg, Non-Veg, or view all items
- **Search Functionality**: Search for dishes by name or ingredients
- **Modern UI**: Clean, professional design with smooth animations
- **Contact Integration**: WhatsApp and phone contact buttons
- **Auto-refresh**: Manual refresh button to sync latest data

## 🚀 Quick Start

1. **Clone or Download** the project files
2. **Open** `index.html` in your web browser
3. **Enjoy** the restaurant dashboard!

## 📊 Data Source

The dashboard automatically fetches data from this Google Sheet:
```
https://docs.google.com/spreadsheets/d/19urkakgRjgueR7eyzKnUsOFUXZNOsmDDXAzM0kFvYN8/export?format=csv&id=19urkakgRjgueR7eyzKnUsOFUXZNOsmDDXAzM0kFvYN8&gid=840320789
```

### Data Structure
The sheet should contain these columns:
- `Item Name`: Name of the dish
- `Quantity`: Available quantity
- `Ingredients`: List of ingredients
- `Price (INR)`: Price in Indian Rupees
